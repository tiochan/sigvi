Final Absolute Severity (FAS) form

This form is used to define the FAS functions to use in order to determine the impact of the vulnerability on each service.

By default, one FAS is enabled for all groups (those without group), but each group can define their own.
Only one can be enabled for each group.

The order to select the FAS function is:

  • First: if the group owner of the server has one FAS enable, will use it
  • Second: else, if there is one without group enabled (generic), will use it
  • Last: else, will use the generic FAS ()
